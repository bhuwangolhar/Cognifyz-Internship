
# 🎓 Cognifyz Project

## 📌 Tasks Overview

### 🔹 Task 1 – Simple UI Element  
Designed a basic UI button focusing on alignment, spacing, color contrast, and readability.

### 🔹 Task 2 – Basic Color Palette  
Created a clean color palette showcasing primary and supporting colors with proper hex codes and visual clarity.

### 🔹 Task 3 – Landing Page
Designed a minimal landing page layout with a clear heading, description, and call-to-action button, focusing on visual hierarchy.

### 🔹 Task 4 – Icon Design  
Designed a scalable sun icon using basic geometric shapes, ensuring balance, symmetry, and clarity.

### 🔹 Task 5 – Form Redesign  
Redesigned a user registration form with properly aligned input fields, labels, and a primary action button for better usability.

### 🔹 Task 6 – Navigation Bar  
Designed a clean and responsive navigation bar with a logo, menu items, spacing consistency, and an active state.

### 🔹 Task 7 – Mood Board  
Created a travel app mood board showcasing inspiration images, color palette, and typography selection to define visual direction.

### 🔹 Task 8 – Microinteraction
Designed a button microinteraction showing different states (default, loading, success) to demonstrate user feedback and interaction flow.
